# ⚇ ddot.it Plain Text Reader

A ddot.it [reader](developer-guide.md#reader) for plain text files (`.txt` and similar).
It recognizes triples in [plain text](#plain-text).

NOTE: Some examples use the [ddot.it/this](user-guide.md#this-command) command.

## Standard Syntax

### Plain Text
Any line in a plain text file can contain ddot.it triples.

```
My notes file.
ddot.it/this ..project.. Code Red
..author.. Alice
Project Alpha .. depends on .. Project Beta
Project Alpha .. status .. active
```
