# ddot.it
ddot.it - just double dot it to connect anything with everything 

Double dot (ddot) is a simple syntax you can use wherever you can enter text. Designed for mobile keyboards, too. 

,,,ddot
double dot..abbreviation.. ddot
..url.. ddot.it
..author.. Max Völkel
....wiki syntax
,,,
fires these triples:

,,,json
[
{"from": "double dot", "type":"abbreviation", "to":"ddot"},

{"from": "double dot", "type":"url", "to":"ddot.it"},

{"from": "double dot", "type":"author", "to":"Max Völkel"},

{"from": "double dot", "type":"links to", "to":"wiki syntax"}
]
,,,
